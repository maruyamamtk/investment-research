① 週次 監視対象銘柄の選定

毎週日曜 8:00 / 全1,600社 → 上位20社

  • Step1: 時価総額・営業利益率・PBR・自己資本比率でまず除外
  • Step2-A/B: 年次・四半期EPS成長 ≥25%（J-Quants財務諸表）
  • Step2-C: 売上高成長（年次2期連続プラス + 四半期3期連続OR最新25%以上）
  • Step2-D/E/F: ROE>15% / FCF品質 / 純負債/EBITDA ≤3.0x
  • Step2-G: 7指標スコアリング（ROE・CAGR・FCF・PEG比率・負債・利益率・配当）で上位20社
  • Gemini AI: Top5銘柄の定性分析（Q1〜Q5）・投資テーゼ・ベアケース
出力: output/watch_list.md
　　　output/weekly_moat_stocks.md（watch_list.md への後方互換シンボリックリンク）
パイプライン: pipelines/weekly_pipeline.py
② 日次 購入銘柄の選定

平日 19:30 / 監視対象リスト → 購入候補リスト

  • 市場レジーム判定（日経225）→ BULL/NEUTRAL/BEAR でBUY閾値を3〜5点に動的変更
  • ①監視対象リストの全銘柄にテクニカル分析を実施
  • チャートパターン認識（ブレイクアウト/ダブルボトム/ブルフラッグ/カップ&ハンドル 等）でスコア加算
  • ゴールデンクロス / RSI売られすぎ反転 / MACDプラス転換 でBUYスコア加算
  • BUYスコア ≥BUY閾値 → 購入候補リストに追加（重複除外・最新シグナル日時更新）
  • 決算前後3日 / データ不足 → HOLD抑制（追加しない）
  • ※チャート状況・市場環境によっては長期間選定が行われない可能性あり
出力: output/buy_candidates.md（BUY時のみ追加）
　　　output/daily_trade_signals.md（当日の全銘柄シグナル一覧）
　　　output/signals.csv（機械可読シグナルデータ）
パイプライン: pipelines/daily_pipeline.py
③ 日次+週次 売却銘柄の選定

平日 19:30（テクニカル）+ 日曜（ファンダメンタルズ）

  • 軸A（日次）: SELLスコア ≥3点 → 即時除外推奨
  • 軸B（週次）: ①と同等のファンダメンタルズ再確認。条件劣化 → 要注意フラグ → 2回連続で除外
  • 両軸同時該当 → 即時除外 + 優先度高のLINE通知
  • 決算前後3日はテクニカルSELLを抑制（ファンダメンタルズ劣化による除外は実施）
処理: output/buy_candidates.md から削除 / LINE通知
パイプライン（軸A・テクニカル）: pipelines/daily_pipeline.py
パイプライン（軸B・ファンダメンタルズ）: pipelines/weekly_pipeline.py
別途 決算レビュー

手動実行 / 監視リスト銘柄の決算Beat/Miss・ガイダンス変化を解析

  • watch_list.md から対象銘柄を取得
  • 各銘柄のEPS・売上実績 vs 予想を比較し Beat / Miss / In-Line を判定
  • ガイダンス変化（上方修正・下方修正・据え置き）を解析
  • LINE通知で結果を送信
出力: output/earnings_review_YYYYMMDD.md
パイプライン: pipelines/earnings_pipeline.py
